Python
======

Python is a programming language that allows us to communicate with our computer to
perform various tasks and solve a wide range of problems. In this module we will
introduce you to the basics of Python

If this is your first time learning how to program, don't worry if you feel overwhelmed
by all the content. Learning to code is hard! You might not understand everything
straight away.

The best way to learn Python is to practice, practice, practice!
